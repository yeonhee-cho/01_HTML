style 태그 내용 적용 확인

block 형식

        - 화면(공간) 수직 분할
        (세로 화면을 가로로 잘라서 총(행) 나눔)

        - 한 층(한 행)을 모두 차지함

        - width, height와 같은 크기 지정 속성 사용 가능

        - block 형식의 태그
        h1 ~ h6(제목), p/pre(문단), hr(수평선), div(영역 분할),
        시멘틱 태그(div 태그에 이름을 넣는 태그)

h3는 block 형식 태그입니다.

        block 형식은 작성된 내용이 짧더라도 한 줄을 모두 차지하려는 성질 떄문에  다음 태그 내용은 다음 줄에 작성됩니다.
    
        pre 옆에 가로로 해서 이어서 작성할 수 없다는 의미로 태그를 닫고 열 때 마다 층을 구분지어서 생성됨

inline 형식

        - 화면을 수평 분할
        (가로 화면을 세로로 잘라서 호(열)을 나눔)

        - 한 층(행)을 모두 차지하지 않음
        -> 내부에 작성된 내용 크기만큼 차지하려는 

        - width, height와 같은 크기 지정 속성 사용 불가능

        - inline 형식의 태그
          b(굵게), i(기울이기), mark(글 배경 칠하기), span(영역 분할) 등

inline 형식은 행을 분할하고 한 열에서 이어서 작성되는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다.
첫 번째 span 두 번째 span 세 번째 span

inline - block 형식

        - inline의 특징 중 수평 분할(줄을 세로로 자름, 열 나눔)
        - block의 특징 중 크기 지정 가능
        장점을 합쳐놓은 태그
        
        - 대표적인 태그 : img 태그, iframe 태그를 
        
        ifram 태그 
        - 웹 문서(html)내부에 또 다른 웹 문서를 추가하는 태그
