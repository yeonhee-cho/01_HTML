표 관련 기본 태그

tabl(표) 태그

        - 웹 문서에서 자료(데이터) 정리를 위해 사용하는 태그
        - table 태그는 행, 열을 작성하기 위한 판을 나타냄

tr(Table Row) 태그

        - 한 행의 한 컬럼을 나타내는 태그
        == 엑셀의 셀 하나

th(Table Header) 태그

        - 테이블의 제목 또는 내용을 강조할 때 사용하는 태그
        - 두꺼운 글씨, 가운데 정렬이 자동으로 적용된다.

caption 태그

        - 테이블에 대한 설명을 작성하는 태그

테이블 관련 추가 태그2

        [테이블 영역 지정 태그]

        thead 태그 : 테이블 상단 영역(컬럼 제목 작성 부분)

        tbody 태그 : 테이블 중단 영역( 내용, 값을 작성하는 부분)

        tfoot 태그 : 테이블 하단 영역(합계, 인원 수)

        (* 참고사항)
        table 태그 내부를 thead, tbody, tfoot 으로 나누지 않으면
        모든 tr, td, th 태그는 tbody 내부에 작성된 것으로 취급된다.

행, 열 병합 속성 : rowspan, colspan

        병합이라는 의미보다 몇 칸을 차지 할 것인지에 대한 의미로 해석
        rowspan : 행 병합(세로 병합)
        colspan : 열 병합(가로 병합)

        table을 자동 완성하는 단축어

        table>tr*3>td*4
        table>thead>tr>th*3
        table>tbody>tr*3>td*4
        tr>td*5{셀 $}
