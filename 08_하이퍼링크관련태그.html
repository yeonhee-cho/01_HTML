a(anchor : 연결, 고정) 태그

        - 현재 HTML문서와 다른 HTML 문서를 연결 시켜 페이지를 이동할 수 있게 만드는 태그

        - a 태그로 감싸진 글/ 내용 == 하이퍼 텍스트(하이퍼 링크)
        - 하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심 기능

a 태그의 속성

        - herf(HyperText PRFerence) : 연결할 주소를 지정하는 속성
        - target = "_blank" : 새 탭에서 열기

        id   는 중복으로 이름 사용을 지양해주는 것이 좋음
